  • 解决 Mac 安装 Homebrew 失败的问题

      • 解决方案二

# 错误日志：curl: (7) Failed to connect to raw.githubusercontent.com port 443: Connection refused

# 解决方案一

  1. 打开网址open in new window，将网址中的内容保存为brew_install.rb。
  2. 在终端中执行ruby brew_install.rb，需要确认请直接按回车。

# 解决方案二

  1. 打开网址open in new window，将网址中的内容保存为brew_install.sh
  2. 在终端中执行/bin/bash brew_install.sh，需要确认请直接按回车。

提示

如果打不开上面的两个网址，请开启科学网络！
